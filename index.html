Welcome to the Structures & Algorithms GitHub Repository

This repository is a working copy of my solutions to the Structures & Algorithms labs. I have released my work to the public domain and you may use it however you choose. I do however make a polite request: please do not directly copy my code, use it for the basis of your own work. I do not guarantee that this code works, fits the solution, or even compiles, but it is an up-to-date copy of what I have done so far. I try to keep this repo as up to date as I can with the latest labs, but I may be a bit behind if there's new labs to do, check back in a few days if there's not a solution up yet.

Downloading as a zip

You can download all of my code so far as a zip (or .tar.gz if that's your thing) from the link at the top right. Or you can just click here!

Previewing code online

You can preview my code on GitHub directly.

Checking out code from GitHub

You can check out my code using the git client with the following code in the command prompt:

> cd %USERPROFILE%\Documents
> git clone https://github.com/chartax/structures-and-algorithms

If you're using a Git GUI client, you can also check out the code through that, or even through Eclipse (or another Java IDE) using a plugin.

Want to comment or add another solution?

Support or Contact
